# Backend & Ngrok Setup Guide

This project now includes a Python backend that acts as a proxy to HuggingFace for translation services. This allows you to hide your API key and gives you control over the requests.

## 1. Setup Backend

1.  Navigate to the `backend` folder:
    ```bash
    cd backend
    ```
    // should be ins in a virtual environment
2.  Install dependencies (recommended to use a virtual environment):
    ```bash
    pip install -r requirements.txt
    ```
    *Note: This now installs `huggingface_hub` for better API handling.*

3.  Run the server with your HuggingFace API Key:
    
    **Windows (PowerShell):**
    ```powershell
    $env:HF_API_KEY="your_huggingface_api_key_here"
    uvicorn app:app --reload --port 8000
    ```

    **Linux/Mac:**
    ```bash
    export HF_API_KEY="your_huggingface_api_key_here"
    uvicorn app:app --reload --port 8000
    ```

    The backend will start at `http://localhost:8000`.

## 2. Expose with Ngrok (Optional)

If you need to access the backend from a deployed frontend or a different device, you can use ngrok or pinggy.

1.  **Install ngrok** (if you haven't): [https://ngrok.com/download](https://ngrok.com/download)
2.  Run ngrok on port 8000:
    ```bash
    ngrok http 8000
    ```
3.  Copy the HTTPS URL generated by ngrok (e.g., `https://a1b2-c3d4.ngrok-free.app`).

## 3. Configure Frontend

1.  Run the React app:
    ```bash
    npm run dev
    ```
2.  Open the website in your browser.
3.  Click the **Translator** icon (top right or floating).
4.  At the bottom of the translator popup, click **⚙️ Cấu hình Server**.
5.  Paste your backend URL (e.g., `https://a1b2-c3d4.ngrok-free.app` or `http://localhost:8000`).
6.  The app will now use this URL for translations.
